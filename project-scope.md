# Project Scope - 个人介绍首页

## 项目概述

一个用于介绍自己的个人网站首页，目标受众为潜在雇主、合作伙伴和感兴趣的访客。作为个人品牌和在线简历展示。

## 技术栈

- **构建工具**：Vite
- **前端框架**：React
- **语言**：仅中文（zh-CN）
- **样式方案**：Tailwind CSS

- **部署**：GitHub Pages


## 功能范围

### 必须包含（Must Have）

1. **个人简介板块** - 头像、姓名、一句话定位、简短自我介绍
2. **技能展示板块** - 掌握的技术栈、工具、软技能等
3. **项目作品展示板块** - 参与或完成的代表性项目
5. **联系方式/社交链接** - GitHub、邮箱、LinkedIn、个人社交账号等
6. **响应式设计** - 适配桌面端、平板、移动端。手机端优先考虑。
7. **暗色/亮色模式切换** - 支持主题切换，记录用户偏好

### 应该包含（Should Have）

1. **页面滚动动画** - 少量、克制的进入动画
2. **导航栏** - 顶部导航，支持快速跳转到各板块

### 可以包含（Could Have）

1. **页面访问计数器**

### 不包含（Won't Have）

1. 用户注册/登录系统
2. 评论/留言功能
3. 后台管理系统
4. 数据库持久化
5. 国际化多语言支持（仅中文）
6. 复杂的交互动画或粒子效果

## 设计风格

- **主风格**：简洁现代（Minimalist）
- **色彩**：大量留白，克制配色
- **排版**：清晰的层级结构
- **视觉**：内容优先，避免装饰性元素

## 页面数量

- **单页应用（SPA）**：1 个主页，包含所有板块
- 不包含独立的子页面（项目详情页、文章详情页后续可扩展）

## 非功能性需求

- **性能**：首屏加载 < 2s，Lighthouse Performance ≥ 90
- **可访问性（A11y）**：语义化 HTML、合理的对比度
- **SEO**：基础的 meta tags、Open Graph tags
- **浏览器兼容**：最新 2 个版本的 Chrome / Edge / Firefox / Safari
